Customer REST API Service for E-Commerce website

GET /customers to see all the customers.

GET /customers?email=abc@xyz.com to see all customers with registered with a particular email.

GET /customers/{id} retrieves a customer from the DB using an ID.

PUT /customers/{id} updates a customer using an ID.

DELETE /customers/{id} removes a customer from the DB using an ID.

POST /customer creates an new customer.

Example Data:

       "first_name": "Jason",
